PROJECT_02 📋 Log Analysis & Alert Triage

Phishing Campaign — Log Analysis & Alert Triage

// Scenario

Simulated a phishing campaign where a user clicked a malicious link. Task: reconstruct the full attack chain from email gateway logs, proxy logs, DNS queries, and endpoint process events.

// Approach

  • Parsed email gateway logs — identified sender spoofing vector
  • Correlated proxy logs to trace outbound C2 domain request
  • Analyzed DNS logs for domain age and TTL anomalies
  • Reviewed Sysmon Event ID 1 for browser child process spawning
  • Mapped attack to MITRE ATT&CK: T1566, T1059

// Tools

Splunk Sysmon Wireshark VirusTotal URLScan.io ATT&CK Navigator

// Key Findings

  • Spoofed domain registered 2 days before the attack
  • HTTP POST beacon attempt to external C2 IP traced
  • cmd.exe spawned as child of chrome.exe — confirmed IOC
  • Formal incident report produced with remediation steps
OUTCOME — Full attack chain reconstructed in <45 min. Written incident report with IOC timeline, MITRE mapping, and containment steps.

PROJECT_04 🔍 Threat Hunting

Threat Hunting — Detecting Lateral Movement with Sysmon

// Scenario

Proactive threat hunt across endpoint Sysmon logs. Hypothesis: attacker with initial access using living-off-the-land binaries (LOLBins) for lateral movement or persistence.

// Approach

  • Defined hypothesis based on MITRE T1021 Lateral Movement
  • Queried Sysmon logs for PsExec, WMI, and PowerShell Remoting
  • Identified anomalous parent-child process chains (Event ID 1)
  • Searched for unauthorized scheduled task creation (Event ID 4698)
  • Converted confirmed TTPs into new SIEM detection rules (Sigma)

// Tools

Elastic SIEM Sysmon KQL MITRE ATT&CK Sigma Rules PowerShell

// Key Findings

  • PowerShell executing encoded commands at 3am on 2 hosts
  • Scheduled task created by unauthorized user account
  • All findings mapped to ATT&CK sub-techniques
  • 2 Sigma rules deployed to SIEM from hunt findings
OUTCOME — 2 persistence mechanisms uncovered, 2 new Sigma rules deployed. Hunt methodology documented as repeatable runbook.
